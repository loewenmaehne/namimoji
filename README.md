## ✨ &nbsp; The idea

Your phone buzzes. You look.

**Namimoji makes that unnecessary.**

It turns vibration into a language your skin can read — structured messages carrying *who* and *what*.

## 🌐 &nbsp; An open standard

Namimoji is an open **standard** for tactile communication, like Unicode for text.

A message encoded on one device can be felt on another. Built for smartwatches, phones, wearables, accessibility tools, and hobby hardware.

**One standard. Many implementations.**
